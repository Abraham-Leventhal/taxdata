About puf_stage3
================

This directory contains the following scripts:

* Python script **stage3.py**, which reads/writes:

  Input files:
    - stage3_targets.csv
    - ../puf_data/cps-matched-puf.csv
    - ../stage1/growfactors.csv
    - ../puf_stage2/puf_weights.csv

  Output files:
    - puf_ratios.csv
